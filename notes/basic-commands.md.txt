Navigation Commands
1.1 pwd = Shows the full path of the directory you are currently in.
1.2 ls = Lists files and directories in the current location.
1.3 ls -lah = Lists files with details, including hidden files, in humanâ€‘readable format.
1.4 cd <directory> = Changes the current directory.
1.5 cd .. = Moves up one directory level.
1.6 cd ~ = Returns to your home directory.

File and Directory Operations
2.1 touch <file> = Creates an empty file.
2.2 mkdir <folder> = Creates a new directory.
2.3 mkdir -p <path> = Creates nested directories without errors if they already exist.
2.4 cp <source> <destination> = Copies a file.
2.5 mv <source> <destination> = Moves or renames a file.
2.6 rm <file> = Deletes a file permanently.

Viewing Files
3.1 cat <file> = Displays the entire contents of a file.
3.2 less <file> = Opens a scrollable viewer for large files.
3.3 head -n 20 <file> = Shows the first 20 lines of a file.
3.4 tail -n 20 <file> = Shows the last 20 lines of a file.
3.5 nl  <file> = Displays a file with line numbers added automatically.
Useful when referencing specific lines.
3.6 more <file > = Shows a file one page at a time, similar to less but more basic.
