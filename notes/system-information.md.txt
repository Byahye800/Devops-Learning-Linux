System Information

These commands give you a quick overview of your system’s hardware, storage, memory, and general configuration. They’re useful for troubleshooting and understanding what resources your system has available.

1: uname -a = Shows system and kernel information.
2: df -h = Displays disk usage in a human‑readable format.
3: du -sh <directory> = Shows the total size of a directory.
4: free -h = Displays memory and swap usage.
5: hostname = Shows the system’s hostname.
6: uptime = Shows how long the system has been running.
7: whoami = Displays the current logged‑in user.
8: lscpu = Shows CPU details such as cores and architecture.
